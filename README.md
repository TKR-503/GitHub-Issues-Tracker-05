2️⃣ What is the spread operator (...)?

Ans: The spread operator is used to expand elements of an array or object into another array or object .
     For the array it is used copies the values and for object it is used copies the properties.

     It is commonly used for copying ,merging and passing multiple values easily .



3️⃣ What is the difference between map(), filter(), and forEach()?

Ans: map() method creates a new array by changing each elements based on another function .

     filter() method creates a new array by keeping only elements that match a condition .
     for each() method just loops through the array without returning anything.
